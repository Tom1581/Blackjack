## Short Description
Practice blackjack strategy with a built-in Hi-Lo count trainer.

## Full Description
Hi-Lo Blackjack Trainer is a fast blackjack practice game built for players who want to sharpen basic strategy, bankroll decisions, and card-counting awareness in a realistic table flow.

Play hands, place bets, track the running count, watch the true count shift as the shoe changes, and practice spotting favorable and unfavorable situations. The game includes common blackjack actions such as hit, stand, double, split, insurance, and side bets, with a clean table interface designed for quick rounds.

Features:
- Built-in Hi-Lo running count and true count display
- Multiple shoe modes, including continuous shuffle, 2-deck, and 6-deck play
- Blackjack actions: hit, stand, double, split, and insurance
- Bankroll tracking and practice-focused table flow
- Weekly local leaderboard and stats history
- Quick portrait gameplay for short training sessions

This app is for entertainment and practice only. It does not offer real-money gambling, real-money prizes, betting with real currency, or the ability to cash out virtual chips.
